about

Hi ! I'm Jonathan Dobson ! I'm a software developer

I began my journey with software in high-school math class, with the introduction of the TI-83 graphing calculator. This wonderful machine included a built-in scripting programming language and a large user manual full of programming instructions.

I knew there were countless opportunities and applications to create software for. So I went through high-school putting my free time towards building various apps for this calculator, and rapidly building my programming skills. During college, I explored software more and discovered the fascinating world of web-based apps.

I starting in the web space with the FrontPage graphical development environment, then quickly moved to software books I obtained from my local public library. Then, I discovered a spectacular book on PHP, so I read it. In just a few months, I turned my PC into an http server, running Apache, a MySQL database, an app that managed inventory, and a domain name pointed to my static IP.

Until today, I've been involved in web software, following the trends, learning, and gaining more skills in web-related tech, such as JavaScript, CSS, React, Linux Terminal, and more.
