about

Hi ! I'm Jonathan Dobson ! I'm a software developer

My enthusiasm for programming apps began in high-school math class. Students were introduced to the TI-83 graphing calculator, an amazing machine which included with purchase, a comprehensive user manual along with it's very own built-in scripting language.

It was at this point, I took interest in the amazing power and the unlimited possibilities of software. I would then go on in high school spending my free time building apps for the calculator, all while rapidly gaining knowlege and skill that I would bring to college. During college, I explored the field of software. I then discovered the amazing and wonderful world of web-based apps.

I entered the internet software world starting off with the FrontPage graphical development environment, then quickly moving on to software books from my local public library. Soon, I discovered a spectacular book on PHP. In just a few months, I had turned my PC into a public http server, running Apache, a MySQL database, an app that managed inventory, and a domain name pointed to my static IP.

From then until today, I've been following the web software world, learning, and gaining more skills in web-related tech, such as JavaScript, CSS, Linux Terminal, and more.
